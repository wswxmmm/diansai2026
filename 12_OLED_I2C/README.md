# 12_OLED_I2C

SSD1306 0.96 inch IIC OLED demo, adapted from the LCKFB MSPM0G3507 module guide.

- OLED controller: SSD1306 compatible 128 x 64 display
- I2C address: 0x3C
- SCL: PA1
- SDA: PA0
- I2C bus speed: 400 kHz

The project includes a small ASCII font and self-contained OLED driver. It displays two test pages with text, numbers, and a border.
